# Example data set: Research Vessel (RV) Survey

An example data set containing spatial information and fish species
catch densities (kg/tow). Not useful for formal analysis!

## Usage

``` r
dat.rv
```

## Format

A data frame with 1371 rows and 11 variables:

- X:

  Easting, UTM NAD83, Zone:20N, Units:km

- Y:

  Northing, UTM NAD83, Zone:20N, Units:km

- longitude:

  decimal degrees, NAD83, EPSG:4269

- latitude:

  decimal degrees, NAD83, EPSG:4269

- year:

  survey year

- depth:

  depth of the tow in meters

- whake.kg.tow:

  catch density of White Hake

- redfish.kg.tow:

  catch density of Redfish

- ahalibut.kg.tow:

  catch density of Atlantic Halibut

- ghalibut.kg.tow:

  catch density of Greenland Halibut

- witch.kg.tow:

  catch density of Witch Flounder
