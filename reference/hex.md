# Hexagonal grid, 10x10km

A shapefile used in the fisheriescape project, and sourced from:
https://gcgeo.gc.ca/geonetwork/metadata/eng/572f6221-4d12-415e-9d5e-984b15d34da4.
CRS = ESRI:102001. Accessed with get_shapefile('hex')

## Usage

``` r
hex
```

## Format

Retrieved as class SpatVector with get_shapefile('hex')
