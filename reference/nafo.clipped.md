# Northwest Atlantic Fisheries Organization (NAFO) shapefile, clipped to land

A shapefile sourced from https://www.nafo.int, restricted to 3Pn4RSTVn.
Accessed with get_shapefile('nafo.clipped')

## Usage

``` r
nafo.clipped
```

## Format

Retrieved as class SpatVector with get_shapefile('nafo.clipped')
