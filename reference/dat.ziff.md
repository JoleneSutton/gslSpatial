# Example data set: Commercial Landings

An example data set containing spatial information and species catch
weights. Not useful for formal analysis!

## Usage

``` r
dat.ziff
```

## Format

A data frame with 4559 rows and 16 variables:

- X:

  Easting, UTM NAD83, Zone:20N, Units:km

- Y:

  Northing, UTM NAD83, Zone:20N, Units:km

- ancfvn:

  Anonymised cfvn

- date.landed:

  date.landed

- species.code:

  species.code

- round.weight:

  round.weight

- main.species.caught:

  main.species.caught

- main.species.sought:

  main.species.sought

- date.caught:

  date.caught

- latitude:

  decimal degrees, NAD83, EPSG:4269

- longitude:

  decimal degrees, NAD83, EPSG:4269

- gear.class:

  gear.class

- year:

  year

- year.landed:

  year.landed

- month.landed:

  month.landed

- day.landed:

  day.landed
